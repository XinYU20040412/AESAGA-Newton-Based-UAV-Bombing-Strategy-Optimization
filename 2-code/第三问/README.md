# 第三问 — 单无人机多弹协同优化

说明：本目录包含单架无人机投放多枚烟幕弹（通常为 3 枚）的多版本实现，包含不同的变异/优化策略与若干改进版（如差分 + L-BFGS 的混合策略）。

关键脚本示例：
- `3.py` / `3.1.py`：主入口或试验脚本。
- `3-初版差分+L-BFGS.py`、`3-2版差分+L-BFGS.py`：使用差分进化 + 局部 L-BFGS 优化的版本。

运行建议：
- 先运行 `3.py` 或 `3.1.py` 做功能验证；若需要更强的搜索或局部加速，请尝试 `3-初版...` 和 `3-2版...`。
- 结果（如遮蔽时长、投放时序）通常会输出为 Excel 或打印在控制台中。
