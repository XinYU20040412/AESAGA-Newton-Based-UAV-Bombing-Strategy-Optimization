# 第三问 — 单机三弹协同时序优化

本目录对应题目问题三：使用 FY1 投放 3 枚烟幕弹，优化投放与起爆时序，使总有效遮蔽时间尽可能长。

## 脚本分层

- `3.py`、`3.1.py`：主流程版本（建议先读这两个）。
- `3-初版差分+L-BFGS.py`：混合优化初版。
- `3-2版差分+L-BFGS.py`：混合优化改进版。

## 模型关注点

- 三枚弹的投放间隔约束（至少 1 秒）。
- 起爆时序与烟幕持续时长（20 秒）耦合。
- 多烟幕云团对遮蔽时间的可不连续叠加。

## 输出产物

- 问题三结果表：`1-paper/附件/result1.xlsx`。
- 中间日志与可视化（具体由脚本配置决定）。

## 运行建议

1. 先用 `3.py` 做小规模调试。
2. 再用 `3-2版差分+L-BFGS.py` 做高质量搜索。
3. 最后核对 `result1.xlsx` 与论文指标一致性。
