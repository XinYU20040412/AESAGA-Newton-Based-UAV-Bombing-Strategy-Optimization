# 第五问 — 多机多弹多导弹综合策略

本目录对应题目问题五：5 架无人机、最多每架 3 枚烟幕弹，对 3 枚来袭导弹进行整体干扰策略求解。

## 两阶段流程

1. 匹配阶段：构建无人机-导弹收益矩阵并完成分配。
2. 局部优化阶段：对每个匹配对求解投放/起爆时序，汇总为全局方案。

## 核心脚本

- `5决策.py`：综合决策主逻辑（若作为入口时优先使用）。
- `5-FY2-end.py`、`5-fy3.py`、`5-FY4-end.py`、`5-FY5-end.py`：分无人机或分场景求解版本。
- `cover_checker.py`：遮蔽判定工具。
- `fitness.py`：适应度计算模块。
- `system_at_t.py`：时刻状态系统。

## 输出产物

- 问题五结果表：`1-paper/附件/result3.xlsx`。
- 若干中间策略日志与调试结果。

## 建议流程

1. 先跑策略分配脚本确认匹配结果。
2. 再分别跑局部优化脚本验证每个匹配对。
3. 最后回填到 `result3.xlsx` 并与论文结论核对。
