# 第五问 — 多机多弹对多导弹的两阶段优化

说明：本目录实现两阶段策略：第一阶段通过熵权法构建综合评分并使用匈牙利算法完成无人机—导弹匹配；第二阶段对每个匹配对调用问题三的局部优化模型以得到遮蔽时长局部最优，最后合并得到全局策略。

主要脚本示例：
- `5-FY2-end.py`, `5-FY3.py`, `5-FY4-end.py` 等。  
- `cover_checker.py`, `fitness.py`, `system_at_t.py` 为通用工具函数（被多个脚本调用）。

运行建议：
- 该问题为分阶段流程，建议先运行匹配模块（若存在独立入口），确认分配矩阵；随后对任意一对（无人机—导弹）调用问题三的脚本以验证局部最优化流程。
