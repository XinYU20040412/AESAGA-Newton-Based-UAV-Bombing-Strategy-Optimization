# 第五问 - 不同方法实验

本目录用于对比第五问在不同策略与不同参数下的性能表现，服务于方法选择和消融分析。

## 常见实验脚本

- `5_2匈牙利.py`、`5_3_匈牙利.py`：匹配策略类实验。
- `5_FY3.py`、`5-FY2.py`、`5-FY4-end.py`、`5-FY5-3.py`：分场景优化实验。
- `5.统一优化.py`：统一求解思路实验。
- `示意图.py`：实验可视化辅助脚本。

## 目录定位

- 面向“方法对比与消融”，不直接作为最终结果提交入口。
- 推荐与 [../第五问/README.md](../第五问/README.md) 配合阅读。

## 复现实验建议

1. 固定随机种子，避免不同方法比较失真。
2. 统一迭代预算（代数、种群、评估次数）。
3. 结果统一记录到同一表格，便于横向分析。
