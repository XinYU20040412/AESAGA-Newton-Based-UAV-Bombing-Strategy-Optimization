# 第一问 — 单无人机单弹模型

说明：本目录实现单无人机投放一枚烟幕弹，计算在给定参数条件下（例如投放时间、起爆延迟、无人机速度与方向）导弹视线被遮蔽的总时长。

主要脚本：
- `1.py`：入口脚本。默认参数在脚本顶部设置（如 `tdrop`, `texpl`, `v`, `theta`），运行后会在控制台输出遮蔽时长并可生成可选图形。

如何运行：
```powershell
python "第一问/1.py"
```

说明与提示：
- 若需修改参数，请在 `1.py` 中查找脚本顶部的参数区进行调整。  
- 结果文件将以脚本内部配置为准，若脚本写入 Excel，请注意输出路径。
