# 第一问 — 单无人机单弹基线计算

本目录对应题目问题一：在给定速度、航向、投放时刻和起爆时刻的条件下，计算烟幕干扰对单枚导弹的有效遮蔽时长。

## 主要脚本

- `1.py`：问题一求解主脚本。

## 输入参数（脚本内）

- 无人机初始位置（FY1）。
- 导弹初始位置与速度。
- `v`：无人机飞行速度。
- `theta`：无人机飞行方向角。
- `tdrop`：投放时刻。
- `texpl`：起爆时刻。

## 输出结果

- 控制台输出有效遮蔽时长。
- 部分版本会输出中间轨迹或图形文件（以脚本实现为准）。

## 运行方式

```powershell
python "2-code/第一问/1.py"
```

## 作用定位

问题一是问题二到问题五的物理基线，后续优化问题都建立在这一遮蔽判定模型上。
