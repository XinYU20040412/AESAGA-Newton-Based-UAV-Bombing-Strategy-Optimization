# 第四问 — 三机协同投放优化

本目录对应题目问题四：FY1、FY2、FY3 各投放 1 枚烟幕弹，对 M1 实施协同干扰并最大化总有效遮蔽时间。

## 主要脚本

- `4源代码.py`：主流程版本。
- `4-1添加轮次读取的优化.py`：增加轮次控制与过程读取。
- `4-2大变异.py`：强化变异策略版本。
- `4-3.py`：后续迭代版本。

## 关键难点

- 三机飞行参数同时优化，搜索空间显著扩大。
- 多烟幕时空叠加与遮蔽区间拼接问题。
- 计算效率和可行解命中率平衡。

## 输出产物

- 问题四结果表：`1-paper/附件/result2.xlsx`。
- 过程日志与中间统计（以脚本配置为准）。

## 运行示例

```powershell
python "2-code/第四问/4源代码.py"
```

建议先降低迭代规模做 smoke test，再跑完整参数。
