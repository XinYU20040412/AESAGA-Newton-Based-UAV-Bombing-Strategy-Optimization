# 参考文献

本目录用于存放论文写作与模型构建过程中引用的文献资料、方法来源与背景材料。

## 建议整理方式

- 按主题分类：优化算法、弹道与动力学、可视化与工程实现。
- 保留文献的完整书目信息（作者、标题、年份、来源）。
- 若引用到论文正文，建议在文献文件名或 README 中标注对应章节。

## 与代码的对应关系

- 问题二（AESAGA）相关文献可映射到 `2-code/第二问`。
- 问题三至问题五（协同优化）相关文献可映射到 `2-code/第三问` 至 `2-code/第五问`。

## 对外展示建议

若需公开仓库，可将关键文献单独列出摘要，便于评审快速理解方法来源。
