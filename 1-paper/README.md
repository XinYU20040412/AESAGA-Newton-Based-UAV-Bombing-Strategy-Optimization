# 1-paper 文档目录

本目录用于存放论文正文、题面原文与结果附件，是对外展示中最核心的文档入口。

## 文件导航

- [主论文 PDF](25年数模国赛论文.pdf)
- [A 题题面 PDF](A题.pdf)
- [附件结果说明](附件/README.md)

## 建议阅读顺序

1. 先看主论文，快速了解问题定义、建模假设和最终结论。
2. 再看题面原文，对照论文中的符号和约束条件。
3. 最后看附件结果表，核对问题三到问题五的关键输出。
