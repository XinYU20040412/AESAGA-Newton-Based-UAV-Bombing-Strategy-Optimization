# 附件结果目录

本目录存放论文对应的结果表，用于支撑问题三、问题四、问题五的最终数值结论。

## 文件说明

- [result1.xlsx](result1.xlsx)：问题三（单机三弹协同）结果表。
- [result2.xlsx](result2.xlsx)：问题四（三机协同）结果表。
- [result3.xlsx](result3.xlsx)：问题五（多机多弹综合策略）结果表。

## 使用方式

- 对外展示时，可在仓库首页直接跳转下载这些表格。
- 论文答辩时，可将表中关键指标与 README 摘要表进行逐项对照。

## 版本管理建议

- 若更新了结果，请保留旧版本并在文件名中增加日期后缀。
- 结果表字段建议保持一致，便于自动比对和复现实验。
